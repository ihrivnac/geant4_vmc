40 /// Detector construction class to define materials and geometry.
41 /// The calorimeter is a box made of a given number of layers. A layer consists
42 /// of an absorber plate and of a detection gap. The layer is replicated.
43 ///
44 /// Four parameters define the geometry of the calorimeter :
45 ///
46 /// - the thickness of an absorber plate,
47 /// - the thickness of a gap,
48 /// - the number of layers,
49 /// - the transverse size of the calorimeter (the input face is a square).
50 ///
51 /// In addition a transverse uniform magnetic field is defined
52 /// via G4GlobalMagFieldMessenger class.
