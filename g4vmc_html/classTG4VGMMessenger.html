Detailed Description

Messenger class that defines commands for VGM XML Exporter.

Implements commands:

  • /vgm/generateRoot
  • /vgm/generateAGDD [lvName]
  • /vgm/generateGDML [lvName]
  • /vgm/setAssembliesInNames bool
  • /vgm/setNameSeparator bool
